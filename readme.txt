== Description ==
O plugin "UEPA Acessível" foi desenvolvido pela Diretoria de Processamento de Dados da UEPA para melhorar a acessibilidade em sites WordPress, com foco na comunidade acadêmica da UEPA. Ele oferece funcionalidades que ajudam a tornar o conteúdo mais acessível para todos, incluindo a leitura de textos, aumento de fonte, alto contraste e muito mais. A ferramenta foi projetada para garantir que o conteúdo da página seja acessível para pessoas com diferentes necessidades.

Funcionalidades:
- Ativação e desativação de leitura de textos selecionados na página.
- Ajuste de tamanho de fonte para facilitar a leitura.
- Alternância entre alto contraste e contraste negativo.
- Alteração para tons de cinza.
- Substituição de fontes para fontes legíveis.
- Acessibilidade com suporte a leitura no iOS e Android.

== Installation ==
1. Faça o upload do plugin "UEPA Acessível" para o diretório `wp-content/plugins/`.
2. Ative o plugin na tela de Plugins do WordPress.
3. Após ativar, o menu de acessibilidade aparecerá, permitindo que os usuários personalizem as configurações conforme necessário.

== Frequently Asked Questions ==

= O que é o "UEPA Acessível"? =
"UEPA Acessível" é um plugin desenvolvido para melhorar a acessibilidade em sites, com recursos que ajudam a tornar o conteúdo mais legível e acessível a pessoas com necessidades especiais.

= Como ativar a leitura de texto? =
Clicando no botão de acessibilidade localizado na lateral superior esquerda da tela, basta clicar no botão para ativar a leitura. Quando a leitura estiver ativada, o texto selecionado será lido em voz alta.

= Como alterar o contraste? =
Você pode alternar entre as opções de alto contraste e contraste negativo diretamente no menu de acessibilidade do plugin.
